## What's challenging about Markdown

It's not so much a challenge, but more tedious, if anything, and that is having to line up a sub-list to a current list by putting dashes or asterisks at the first character of the top list. It's a little detail I can see myself forgetting, and it will cause me to have to go back and edit this error in my work, slowing my workflow and probably annoying me just a tad every time I catch myself forgetting.

## What's interesting about Markdown

The coolest thing I found out is that Google Docs supports Markdown and inline code. I never knew that Markdown was so widely used that it was even implemented in software as well-known as Google Docs.
